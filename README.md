## 🎯 Objetivo e Regras do Projeto
O desenvolvimento seguiu rigorosamente as diretrizes do sistema de avaliação da disciplina:
- **Zero Templates:** Estrutura HTML e estilização CSS desenvolvidas 100% do zero.
- **Estrutura de 5 Páginas:** Criação de um ecossistema completo com navegação interligada.
- **Publicação:** Hospedagem em nuvem acessível via web.

---

## 📂 Estrutura das Páginas

1. **`index.html` (Home):** Apresentação principal com resumo de habilidades e foco em Data Science e IA.
2. **`sobre.html` (Sobre Mim):** Biografia detalhada, histórico acadêmico e grid de competências técnicas.
3. **`portfolio.html` (Projetos):** Vitrine de repositórios, destacando análises financeiras com Python, orquestração de LLMs com LangChain e fundamentos de controle de versão.
4. **`novidades.html` (Artigos/Blog):** Linha do tempo de publicações e reflexões sobre o aprendizado na área tech, com páginas individuais para leitura imersiva (`artigo-ia.html`, `artigo-faculdade.html`, `artigo-web.html`).
5. **`contato.html` (Contato):** Interface de comunicação e links de direcionamento profissional.

---

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Semântica estrutural para garantir acessibilidade e organização do conteúdo.
- **CSS3:** Estilização moderna utilizando variáveis (Custom Properties), Flexbox e CSS Grid para layouts responsivos.
- **UI/UX:** Design focado no padrão "Dark Mode", proporcionando uma leitura confortável e um visual alinhado ao mercado de tecnologia.
